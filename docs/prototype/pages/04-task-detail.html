背景

现有登录页样式陈旧，且体验较差。我们想参考 Linear 风格 做一次重设计，统一公司内部多个产品（CRM、工单系统、演示站）的登录入口。

需要支持

  • 手机号 + 验证码登录（主）
  • 邮箱 + 密码登录（次）
  • SSO 跳转入口（公司内部 SSO）
  • 第三方登录暂不做，留接口位

设计要求

  1. 整体风格参考 Linear，简洁、现代
  2. 主色 #5e6ad2，与公司品牌色一致
  3. PC + Mobile 两套适配，Mobile 优先
  4. 登录成功后跳转回原页面
参考：Linear / Vercel / Notion 登录页

验收点

  • 手机号格式校验
  • 60s 验证码倒计时
  • 错误提示文案友好
  • 键盘可达性（Tab 顺序合理）
