取消任务

不同状态取消规则不同 · 以下展示 DRAFT / OPEN / ASSIGNED 三种

OPEN 或 DRAFT

单方取消

  • 发布者 自己取消即可
  • 不需要任何确认
  • 投标记录自动置 WITHDRAWN
  • 状态 → CANCELLED（终态）
ASSIGNED

需双方同意

  • 任一方发起取消
  • 对方需同意
  • 可填「已做工时折算」备注
  • 状态 → CANCELLED
COMPLETED

不可取消

  • COMPLETED 是终态
  • 如有错误，找双方+老板处理
  • 记录进 Change Log 即可追溯
  • 无「取消」按钮
