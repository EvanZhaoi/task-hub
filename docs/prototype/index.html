内部任务交易平台 · Linear-inspired 风格 · 共 16 个页面 · 点击进入查看

如何使用： 每个页面是独立的 HTML 自包含文件（无 JS 依赖、可单独打开）。 页间用「下一张 / 上一张」浮动按钮串联。所有页面共用 assets/style.css。 Mock 数据命名全部带 MOCK_ 前缀，零真实业务信息。
